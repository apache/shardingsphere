## SPI 接口

| SPI 名称         | 详细说明     |
| ---------------- | ------------ |
| EncryptAlgorithm | 数据加密算法 |

## 示例

### EncryptAlgorithm

| *已知实现类*         | *详细说明*             |
| ------------------- | --------------------- |
| MD5EncryptAlgorithm | 基于 MD5 的数据加密算法 |
| AESEncryptAlgorithm | 基于 AES 的数据加密算法 |
| RC4EncryptAlgorithm | 基于 RC4 的数据加密算法 |
| SM4EncryptAlgorithm | 基于 SM4 的数据加密算法 |
| SM3EncryptAlgorithm | 基于 SM3 的数据加密算法 |
