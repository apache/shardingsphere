## Definition

```sql
SHOW RESOURCES [FROM schemaName]  
```

## Description

| Column        | Description             |
| ------------- | ----------------------- |
| name          | Data source name        |
| type          | Data source type        |
| host          | Data source host        |
| port          | Data source port        |
| db            | Database name           |
| attribute     | Data source parameter   |
