## 定义

```sql
SHOW RESOURCES [FROM schemaName]  
```

## 说明

| 列        | 说明      |
| --------- | -------- |
| name      | 数据源名称 |
| type      | 数据源类型 |
| host      | 数据源地址 |
| port      | 数据源端口 |
| db        | 数据库名称 |
| attribute | 数据源参数 |

## 示例

```sql
mysql> show resources;
```
