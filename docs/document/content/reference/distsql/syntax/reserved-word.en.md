## Reserved Word

---

### Data source name reserved words

```sql
SYS, MYSQL, INFORMATION_SCHEMA, PERFORMANCE_SCHEMA
```

### Standard reserved words

```sql
ADD, RESOURCE, HOST, PORT, DB, USER, PASSWORD, PROPERTIES, URL
```

### Supplement

- 数据源名保留字影响资源、读写分离规则、高可用规则已经影子库规则
- 标准保留字影响所有资源和规则的变量命名
- 上述保留字大小写不敏感
