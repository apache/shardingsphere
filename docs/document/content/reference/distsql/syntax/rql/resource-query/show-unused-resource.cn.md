### 描述

`SHOW UNUSED RESOURCE` 语法用于查询指定逻辑库中还未被规则引用的资源。

### 语法

```sql
ShowUnusedResource ::=
  'SHOW' 'UNUSED' 'DATABASE'? 'RESOURCES' ('FROM' databaseName)?

databaseName ::=
  identifier
```

### 特别说明

- 未指定 `databaseName` 时，默认是当前使用的 `DATABASE`； 如未使用 `DATABASE` 则会提示 `No database selected`。

### 返回值说明

| 列        | 说明      |
| --------- | -------- |
| name      | 数据源名称 |
| type      | 数据源类型 |
| host      | 数据源地址 |
| port      | 数据源端口 |
| db        | 数据库名称 |
| attribute | 数据源参数 |
