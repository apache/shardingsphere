## MD5 加密算法

类型：MD5

可配置属性：无

## AES 加密算法

类型：AES

可配置属性：

| *名称*         | *数据类型* | *说明*        |
| ------------- | --------- | ------------- |
| aes-key-value | String    | AES 使用的 KEY |

## RC4 加密算法

类型：RC4

可配置属性：

| *名称*         | *数据类型* | *说明*        |
| ------------- | --------- | ------------- |
| rc4-key-value | String    | RC4 使用的 KEY |
