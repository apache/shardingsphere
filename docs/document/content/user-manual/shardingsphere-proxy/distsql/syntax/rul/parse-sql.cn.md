### 描述

`PARSE SQL` 语法用于解析 `SQL` 并输出抽象语法树。

### 语法

{{< tabs >}}
{{% tab name="语法" %}}
```sql
ParseSql ::=
  'PARSE' sqlStatement 
```
{{% /tab %}}

### 返回值说明

| 列                       | 说明        |
|-------------------------|-----------|
| parsed_statement        | 解析SQL语句类型 |
| parsed_statement_detail | 解析SQL语句细节 |

### 示例

- 解析 `SQL` 并输出抽象语法树

```sql
PARSE SELECT * FROM t_order;
```

```sql
mysql> PARSE SELECT * FROM t_order;
+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| parsed_statement     | parsed_statement_detail                                                                                                                                                                                                                                                                               |

```
