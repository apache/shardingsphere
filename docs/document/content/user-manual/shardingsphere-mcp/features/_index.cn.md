ShardingSphere-MCP 通过功能插件扩展领域能力。
功能插件如果需要多步骤治理变更，会使用[插件工作流](plugin-workflow/)完成规划、预览、执行和校验阶段。

发行包默认包含以下官方 MCP 功能插件：

- 数据加密：规划、执行和校验数据加密规则。
- 数据脱敏：规划、执行和校验数据脱敏规则。

新增或第三方功能插件可以通过 `plugins/` 目录加入运行时类路径。
如果功能插件未随发行包提供，启动前需要同时准备它依赖的 ShardingSphere 模块和第三方 jar。
